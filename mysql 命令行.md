# mysql 命令行(Mac)

## 安装(待补充)

```bash
brew install mysql
```



## 服务

启动

```bash
mysql.server start
```



停止

```bash
mysql.server stop
```



重启

```bash
mysql.server restart
```



登录

```bash
mysql -u username -p 
```



## 数据库操作

显示所有数据库

```mysql
show databases;
```



切换数据库

```mysql
use database;
```



显示所有表

```mysql
show tables;
```
